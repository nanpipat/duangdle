# Project Specification: Daily Siamsi (เว็บเซียมซีรายวัน)

## 1. Project Overview & Goal

สร้าง Web Application หน้าเดียว (Single Page Application) สำหรับเสี่ยงเซียมซีรายวัน โดยมีเงื่อนไขสำคัญคือ "1 คน / 1 ดวง / 1 วัน" (1 User per 1 Fortune per 1 Day)

- มี Interaction หลักคือการ "คลิกลากเพื่อเขย่ากระบอกเซียมซี" (หรือเขย่าเครื่องบนมือถือ)
- ผลลัพธ์ดวงต้องล็อกตามตัวตนผู้ใช้และวันที่ (Deterministic) กดรีเฟรชหรือเล่นใหม่ในวันเดียวกันต้องได้ใบเดิม
- หน้าผลลัพธ์ดวงเป็น UI กระดาษเซียมซีโบราณแบบยาว เขียนอธิบายร่ายยาวรวมทุกเรื่อง (การงาน การเงิน ความรัก สุขภาพ) ในข้อความเดียวตามสไตล์เซียมซีดั้งเดิม

## 2. Tech Stack (Suggested)

- Frontend: Next.js (App Router) หรือ React + Vite (ตามความเหมาะสมของ Project ปัจจุบัน)
- Styling: Tailwind CSS
- State Management: React Hooks (useState, useEffect)
- Persistence: Browser LocalStorage

## 3. Core Logic: "1 User / 1 Fortune / 1 Day" (Deterministic Seed)

เพื่อไม่ให้ผู้ใช้โกงสุ่มใหม่ และไม่ต้องใช้ระบบ Login/Database ให้ใช้ระบบ Seed-based Randomization:

1. ระบบจะเช็คใน `LocalStorage` ว่ามี `siamsi_user_id` (UUIDv4) หรือยัง ถ้าไม่มีให้แอบสร้างและบันทึกไว้
2. นำ `user_id` มารวมกับ `current_date` (Format: YYYY-MM-DD) เพื่อใช้เป็น Seed ในการคำนวณสุ่ม
3. ใช้ฟังก์ชัน Hash หรือ Seeded Random (เช่น MurmurHash หรือ LCG) เพื่อแปลง Seed นั้นให้กลายเป็น "หมายเลขใบเซียมซีที่ถูกล็อกไว้แล้วของวันนี้" (เช่น ใบที่ 1 ถึง 30)
4. เมื่อเข้าเว็บมา:
   - ถ้าตรวจพบใน `LocalStorage` ว่าวันนี้ผู้ใช้เคยเขย่าไปแล้ว -> ให้ข้ามไปแสดงผลหน้ากระดาษคำทำนายของวันนี้ทันที
   - ถ้ายังไม่เคยเขย่าในวันนี้ -> แสดงหน้าแรก (กระบอกเซียมซี) เพื่อให้ผู้ใช้ทำ Interaction

## 4. UI/UX & Interaction States

### State 1: หน้าเขย่าเซียมซี (The Shake)

- **UI:** แสดงกระบอกเซียมซีตั้งอยู่ตรงกลางหน้าจอ มีไม้ติ้วโผล่ออกมาจากขอบกระบอกหลายๆ ใบ บรรยากาศนวลๆ มินิมอลผสมกลิ่นอายศาลเจ้าโมเดิร์น
- **Interaction (Desktop):** ผู้ใช้ใช้เมาส์ Click ค้างที่กระบอกแล้วลากซ้าย-ขวา หรือบน-ล่าง ไวๆ (Mouse Dragging)
- **Interaction (Mobile):** ใช้การสัมผัส (Touch Move) ลากขึ้นลงไวๆ หรือถ้าเป็นไปได้ให้รองรับ Device Motion API สำหรับการเขย่าเครื่องจริง (พร้อมสั่งสั่นเครื่องสั้นๆ ด้วย Vibration API)
- **Animation:**
  - ขณะเขย่า กระบอกต้องเอียงและสั่นตามแรงเมาส์/การเคลื่อนไหว ไม้ติ้วข้างในกระเด้งขึ้นลงตามจังหวะ
  - เมื่อเขย่าสะสมความแรงครบตามเกณฑ์ (ประมาณ 2-3 วินาที) ให้เล่น Animation "ไม้ติ้ว 1 ใบพุ่งร่วงหล่นลงมาที่พื้นหน้าจอ"
  - บนไม้ติ้วใบนั้นจะสลัก "หมายเลขเซียมซี" (เช่น เลข ๒๔) จากนั้นระบบจะสลับหน้าเข้าสู่ State 2

### State 2: หน้าคำทำนาย (The Reveal Paper)

- **UI:** หน้าจอจะค่อยๆ Fade หรือมี Animation กระดาษเซียมซีสีครีม/ส้มสไตล์วินเทจ แผ่นยาว ค่อยๆ เลื่อน (Slide down) ลงมาจากขอบจอด้านบนเหมือนการดึงกระดาษออกมาจากแผง
- **Content ในกระดาษเซียมซี:**
  - หัวกระดาษ: "เซียมซีใบที่ [เลขไทย]"
  - เนื้อหาคำทำนาย: เป็นความเรียงร่ายยาวฟีลบทกลอนหรือความเรียงดั้งเดิม รวมเรื่องราวชีวิต การงาน การเงิน ความรัก และโชคลาภ ผสมผสานอยู่ในข้อความยาวข้อความเดียว (ไม่แยกหมวดหมู่) ตัวอย่างเช่น:
    > "ใบที่ ๒๔ นี้ตกได้ใบดีเปรียบเหมือนต้นไม้ที่เหี่ยวแห้งได้รับสายฝนดับกระหาย การงานที่ติดขัดจะมีผู้ใหญ่ยื่นมือเข้ามาอุปถัมภ์ชูชุบ ลาภผลเงินทองที่รอคอยย่อมได้สมความปรารถนาในไม่ช้า ส่วนเรื่องความรักนั้นไซร้คนโสดยังต้องร้องเพลงรอไปก่อน ส่วนคนมีคู่ให้ระวังวาจาขัดแย้งกันเล็กน้อย..."
  - ท้ายกระดาษ: มีปุ่ม "แชร์ดวงประจำวัน" และตัวนับ Countdown เวลาที่เหลือก่อนจะเริ่มเสี่ยงทายใหม่ได้ ( reset ทุกเที่ยงคืน 00:00 น. )

## 5. Share Feature (Wordle-Style Gimmick)

🔮 เซียมซีรายวันประจำวันที่ [DD/MM/YYYY]
ฉันเสี่ยงทายตกได้ "ใบที่ ๒๔"
ดวงโดยรวมวันนี้: 🟩🟩🟩🟩⬛ (ดีมาก)
